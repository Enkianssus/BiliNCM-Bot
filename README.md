# **🎵 BiliNCM-Bot (B站网易云全自动点歌机)**

BiliNCM-Bot 是一款专为 Bilibili 主播打造的 **现代化网易云音乐全自动弹幕点歌机**。

彻底告别传统点歌机低效且容易受干扰的“模拟按键”操作。本项目采用「Headless C\# 后端 \+ 现代化 Web 前端」的架构，并开创性地使用了 **CDP (Chrome DevTools Protocol) 底层 JS 注入技术** 直接接管网易云内核，为您带来丝滑、精准且完全静默的点歌体验。

## **✨ 核心特性 (Core Features)**

* 🚀 **CDP 底层注入控制**: 通过浏览器调试协议直接操纵网易云内部 Redux Store，实现毫秒级响应、后台静默切歌、防置顶干扰。  
* 📡 **高精度切歌雷达**: 实时监听网易云底层 TRACK\_CHANGED 事件，解决前端进度不同步、漏歌、卡歌等痛点。  
* 🖱️ **可视化拖拽排歌**: 无需全靠弹幕！通过美观的 Web 控制台，主播可以直接使用鼠标**拖拽调整队列顺序**，甚至将当前播放歌曲拖回队列退尾。  
* 💎 **亚克力玻璃态 UI**: 精心设计的全透明前端，支持完美嵌入 OBS / 直播姬的浏览器源，内置华丽的进入、通知等动画效果。  
* 🔐 **细粒度大航海权限**: 完美适配 B 站直播间生态，支持按**粉丝牌等级**、**航海舰队（总督/提督/舰长）**、**房管**等条件独立分配点歌、切歌、置顶和强控权限。  
* 🔄 **一键热更新**: 内置 Velopack 更新框架，无需离开控制台即可完成版本检查与自动覆盖重启。

## **🚀 快速开始**

### **1\. 安装**

1. 前往项目的 **Releases** 页面下载最新版本的 BiliNCMBot-Setup.exe 或便携版 BiliNCMBot-win-Portable。  
2. 双击运行安装包。

### **2\. 使用步骤**

1. 启动后，程序会在后台静默运行，并自动打开前端控制台 http://localhost:5555/。  
   *(在直播姬或 OBS 中使用【浏览器源】捕捉并输入此网址即可显示在直播画面中)*。  
2. 确保你的电脑上已经正常运行了 **网易云音乐客户端** (需处于登录状态且能正常播放)。  
3. 点击悬浮窗右上角的 **🎛️ (控制台按钮)** 呼出管理面板。  
4. 在面板中选择 **「扫码登录」**，使用手机 B站 APP 扫码以获取弹幕连接凭证。  
5. 在 **「运行状态」** 页面输入你想监控的 B站直播间房间号，点击连接。  
6. 发送弹幕进行点歌，尽情享受全自动的音乐直播吧！

*(注：系统启动时会自动为网易云挂载调试端口并可能重启网易云，此为 CDP 注入必须步骤)*

## **💬 观众弹幕指令**

| 指令格式 | 功能说明 | 默认权限要求 (可在后台调整) |
| :---- | :---- | :---- |
| **点歌 \[歌名\]** | 搜索并添加歌曲到待播队列 | 需一定粉丝牌等级 |
| **置顶(点歌) \[歌名\]** | 优先播放！将歌曲移动到队列最前方 | 需「舰长」及以上 |
| **插队(点歌) \[歌名\]** | 直接中断当前歌曲，强制立刻插播 | 仅房管及超级用户 |
| **立即播放 \[歌名\]** | 从已有队列中找出该歌曲并强制立刻播放 | 仅房管及超级用户 |
| **撤回** 或 **取消** | 从队列中删除自己最近点的一首歌 | 需一定粉丝牌等级 |
| **切歌** 或 **跳过** | 直接跳过当前正在播放的歌曲 | 仅点歌人本人、房管或超管 |
| **移除 \[歌名\]** | 强行从队列中废弃、删除一首歌曲 | 仅房管及超级用户 |
| **开启/关闭点歌** | 直播间全局控制：暂停接收新的点歌请求 | 仅房管及超级用户 |
| **test** 或 **测试** | 测试弹幕机连接状态 | 无限制 |

*(💡 小贴士：点歌时推荐使用 点歌 歌曲名 歌手名 这种格式来避免点到重名的翻唱或伴奏版)*

## **🛠️ 技术栈**

* **后端架构:** C\# / .NET / HttpListener
* **核心交互:** Chrome DevTools Protocol (CDP) WebSocket / Bilibili Live WebSocket API  
* **前端渲染:** React / Vite / Tailwind CSS / 原生拖拽 API  
* **更新框架:** Velopack
